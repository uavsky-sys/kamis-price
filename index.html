🔑 API 키 발급 방법

ℹ️ 2025년 2월부터 aT OpenAPI는
공공데이터포털(data.go.kr)로 이관됨
📌 방법 1 — data.go.kr (aT 경매정보)
① data.go.kr 접속 → 회원가입
② "공영도매시장 경매원천정보" 검색
③ 활용신청 → 즉시 또는 1~2일 승인
④ 마이페이지 → 인증키 확인 후 입력
📌 방법 2 — KAMIS (농산물유통정보)
① kamis.or.kr → 오픈API → 활용신청
② 회원가입 후 인증키 이메일 수신
✅ 둘 다 무료 · data.go.kr 추천
